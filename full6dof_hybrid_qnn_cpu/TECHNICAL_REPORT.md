## 1. Verification Outcome

This folder is **not** an end-to-end 6-joint direct predictor.

- Learned part: shoulder representation for J1-J3 (sin/cos outputs, 6 values).
- Analytical part: J4-J6 recovered from transform decomposition in evaluation.
- Hybrid mechanism: frozen ANN backbone + trainable quantum residual branch.

Therefore, full 6-DOF output is generated by a **learned shoulder + analytical wrist** pipeline.

## 3. Model Architecture

### 3.1 Top-level hybrid equation

$$
\hat{y}_{hybrid} = y_{ANN} + \alpha_{eval}\,\tanh(g)\,W_q\,q(x)
$$

where:

- $y_{ANN}$ is frozen ANN backbone output (6 shoulder sin/cos values).
- $q(x)$ are quantum features (Z expectations per qubit).
- $W_q$ is the residual projector (`Linear(4->6)`).
- $g$ is scalar residual gain.

### 3.2 Frozen ANN backbone

Backbone architecture is loaded from checkpoint and frozen:

- Input: 3
- Hidden width: 384
- Residual blocks: 6
- Output: 6
- Frozen parameters: 1,787,910

### 3.3 TinyQuantumLayer (trainable)

Checkpoint/default configuration:

- Qubits: 4
- Variational layers: 2
- Statevector dimension: $2^4 = 16$

Gate flow:

1. Classical encoder `Linear(3->4)` then angle mapping $\pi\tanh(\cdot)$.
2. Data encoding: one `RY` per qubit.
3. Per variational layer:
   - `RX`, `RY`, `RZ` on each qubit (12 single-qubit rotations/layer).
   - CNOT ring entanglement: `0->1->2->3->0` (4 CNOT/layer).
4. Measurement: Z expectation on each qubit, yielding 4 features.

Totals across 2 layers:

- Single-qubit variational rotations: 24
- CNOT gates: 8

## 4. Training Objective

The hybrid trainer reuses `FullIKLoss`:

$$
L = w_{sc}L_{sc} + w_{wc}L_{wc} + w_{circ}L_{circ}
$$

with default weights:

- $w_{sc}=1.0$
- $w_{wc}=2.0$
- $w_{circ}=0.02$

Terms:

1. $L_{sc}$: SmoothL1 on normalized shoulder sin/cos outputs.
2. $L_{wc}$: FK wrist-center consistency from predicted J1-J3.
3. $L_{circ}$: unit-circle regularization for sin/cos pairs.

Analytical geometry is therefore used inside the optimization objective.

---

## 5. Training Process (Step-by-Step)

1. Load ANN checkpoint and freeze backbone parameters.
2. Build hybrid residual model (`HybridQNNIK`).
3. Create deterministic train/val/test split and normalize wrist-center features.
4. Train with two-group AdamW:
   - quantum branch LR = 3e-3
   - projector + gain LR = 1e-3
5. Use cosine warm-restart scheduler.
6. Each batch:
   - forward
   - compute composite loss
   - skip if non-finite
   - backprop + gradient clipping (1.0)
7. Monitor validation shoulder sin/cos loss.
8. Save atomic `hybrid_last.pt` and best-improvement `hybrid_best.pt`.
9. Early stop when patience condition is met after minimum epochs.
10. Reload best weights.
11. Calibrate deployment alpha on validation grid $[0.0, 1.5]$ with step 0.05.
12. Evaluate hybrid and ANN baseline on the same test split.
13. Save `hybrid_final.pt`, `hybrid_eval_results.npz`, and plots.

This hybrid folder remains a decoupled IK pipeline:

- Shoulder learning is performed by ANN + quantum residual refinement.
